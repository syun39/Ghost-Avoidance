# Ghost Avoidance

このゲームは、こちらに向かってくる幽霊を避けてゴールを目指すゲームです。

## 概要

このゲームは、幽霊に当たるとゲームオーバーで幽霊を避けてゴールまで行けるとゲームクリアです。

- **ジャンル**: ホラー
- **ターゲット**: 全てのプレイヤー
